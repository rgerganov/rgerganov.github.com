Cloning RFID cards

29 Jan 2017, 13:27

This post summarizes my experience with cloning RFID cards that I am using on daily basis. There is nothing new here, just a summary of well-known hacks that I found on the internet.

Corporate badge

At work we use 125KHz passive RFID badges which are easy to clone. Each badge has unique ID, so the first step is to read this ID. I have been using this DIY reader based on an Arduino:

Residential entry

The building where I live (and many other buildings) has a front door which is opened with an NFC tag. I found that it is Mifare Classic 1k tag which has been cracked long time ago. Actually, it turned out that I don’t have to crack anything to create a clone because the tag had default keys. They were relying only on the fact that each tag has unique ID and the door opens when pre-recorded ID is shown. Some fine people in China are selling tags that you can program with whatever ID you want, so creating is a clone a simple matter of running nfc-mfsetuid:
